# AutometaAI Landing Page Components

Modern, animated landing page components for AutometaAI - AI-powered metadata generation and image-to-prompt conversion platform.

## Components

### Navbar
- Fixed navigation with scroll effects
- Mobile responsive with hamburger menu
- Smooth animations with Framer Motion

### Hero
- Full-screen hero section with gradient background
- Animated background elements
- Stats display
- Scroll indicator

### Features
- 6 feature cards with hover effects
- Gradient icons
- Smooth animations on scroll

### HowItWorks
- 3-step process visualization
- Connected flow design
- Animated step cards

### Platforms
- Stock platforms showcase (Adobe Stock, Shutterstock, etc.)
- AI platforms showcase (Ideogram, Runway ML, etc.)
- Dual-column layout with checkmarks

### Stats
- 4 key metrics display
- Animated counters
- Icon-based design

### Pricing
- 3 pricing tiers (Starter, Pro, Enterprise)
- Feature comparison
- Popular plan highlight

### CTA
- Call-to-action section
- Trust indicators
- Animated background

### Footer
- Multi-column layout
- Social media links
- Newsletter signup ready

## Design System

- **Font**: Urbanist (Google Fonts)
- **Color Scheme**: Dark theme with purple/pink gradients
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Tailwind CSS
